# VIRID Inc. Armoury

    A W.I.P. weapon pack for Hideous Destructor. Requires HDCoreLib, HD-BulletLib Recasted.
	
## Currently Added Content:
 - Satya Light cannon
 - Nail ammo & Reloader
 - Nail Pistol
 - Nailgun
 - Queen Scattergun
 - Dumbo Grenade Launcher
 - Sabre
 - Pipe Wrench
 - Cleaver
 - Babuin Spray
 - Burn Heal
 - CO2 Grenades
 - Shield Badges
 - Battery Dynamo
